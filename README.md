# 포털형 CMS

## 개요
- 포털, 홈페이지를 구축할때 템플릿 형식으로 가져다 사용할 수 있는 웹어플리케이션
- 쉽고 편리하게 관리할 수 있는 Admin 을 제공
- 백엔드와 프론트엔드를 분리
- Cloud Native 하도록 설계하여 개발

### 사용기술
- Spring Boot 2
- Spring Data JPA
- Spring Security, JWT
- Mysql 5.7
- Flyway
- jUnit5, RestAssured
- Vue
